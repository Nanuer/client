# Nanuer - client

### About the project

소비를 나누어, 소비를 줄여줄 수 있는 앱

더 알아보기 : https://www.notion.so/kidzero/8d99ca190ca04a08a131eab5dac4b4da

### 개발 동기

UMC 2기에서 여름방학동안 진행한 프로젝트이다. 돈이 비교적 부족한 사회초년생, 대학생들을 위해 배달비, 택시비 등과 같은 작은 소비를 줄여 돈을 절약할 수 있도록 하기 위해 개발을 진행했다.
